# TSI Web Application Development Tools

Laboratory works for the Web Application Development Tools course.

## Repository Structure

This repository contains multiple laboratory works, each in a separate branch:

- **Lab1** - HTML/CSS/JavaScript with Express.js server
- **Lab2** - Django User Authentication System
- **Course_Project** - Expense Tracker Application

## How to Check Laboratory Works

### Lab 1: HTML/CSS/JavaScript with Express.js

To view Lab 1, switch to the `Lab1` branch:

```bash
git checkout Lab1
```

Or view it directly on GitHub:

## Clone and Setup

```bash
# Clone repository
git clone https://github.com/alex96lv/TSI-Web-Application-Development-Tools.git
cd TSI-Web-Application-Development-Tools

# Switch to desired lab branch
git checkout Lab1   # For Lab 1
# or
git checkout Lab2   # For Lab 2
# or
git checkout Course_Project   # For Final Project
```

Each branch contains its own README with specific installation and running instructions.
